# UoP_Year4_Scientific_Computing_and_Simulation
This page has all sources and code related to lecture notes, after-class lab and coursework mini project. This lecture mainly focus on study and analysis the behaviour of how to use computer to solve simulation problem. 

Relating to ：
- Fourier Transform(FT), Inverse Fourier Transform
- FFT
- CT with Radon Transform
- Radio Interferometry with the van Cittert-Zernicke Theorem

- Particle Simulation
- HPP Model and FHP Model

- Excited States

- Lattice Boltmann Mothods (LBM)

Within all these method, we discuss various methods related to parallise the simulation for speed up, using different illustation method to display our results.
